# DataGridX

## Overview

DataGridX is an interactive web-based project built with Vue.js, designed for Selenium users, web scrapers, and automation testers to practice data extraction from dynamic table lists. It provides a realistic dataset containing various product data points, along with interactive elements such as pagination and modal popups for review details.

## Features

- **Product Data Table**: Contains product information with the following data points:
  - SKU ID
  - Brand
  - Description
  - Category
  - Price
  - Discount (%)
  - Rating
  - Return Policy
  - Reviews
- **Interactive Features**:
  - Click and extract reviews from a popup modal.
  - Navigate through pages and extract data dynamically.

## Use Cases

- **Web Scraping Practice**: Ideal for testing and improving web scraping techniques.
- **Selenium Testing**: Useful for automation testers to validate interaction-based test cases.
- **Data Extraction Workflow Development**: Helps in developing and testing automated data extraction workflows for structured tables.

## Technology Stack

- **Frontend**: Vue.js
- **Styling**: Tailwind CSS
- **Backend API**: (https://dummyjson.com/products)

## Installation & Setup

1. Clone the repository:
   ```sh
   git clone https://github.com/yourusername/DataGridX.git
   ```
2. Navigate to the project directory:
   ```sh
   cd DataGridX
   ```
3. Install dependencies:
   ```sh
   npm install
   ```
4. Run the development server:
   ```sh
   npm run dev

## How to Use

1. Load the DataGridX webpage.
2. Use Selenium or web scraping scripts to extract table data.
3. Click on reviews to test modal extraction.
4. Navigate through pages to practice pagination-based scraping.
